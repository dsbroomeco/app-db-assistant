# DB Assistant

A cross-platform desktop database manager for SQL and NoSQL databases.

![Platform Support](https://img.shields.io/badge/platform-Windows%20%7C%20Linux-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## Supported Databases

| Database | Compatible With |
| -------- | --------------- |
| PostgreSQL | 12 – 17 |
| MySQL | 5.7, 8.0, 8.4 |
| MariaDB | 10.6, 10.11, 11.x |
| SQLite | 3.x (any recent version) |
| Microsoft SQL Server | 2019, 2022 |
| MongoDB | 5.0 – 8.0 |
| Redis | 6.2, 7.x |

## Features

**Connections & Security**
- Encrypted credential storage using OS keychain / Electron `safeStorage` — credentials never reach the renderer process
- SSH tunnel support for remote databases
- Configurable SSL/TLS per connection; `sslRejectUnauthorized` toggle on all drivers

**Schema Browser**
- Tree view: databases → schemas → tables → columns, views, and routines
- Table structure inspector (columns, types, indexes, constraints)
- Schema diff — compare schemas between two connections side by side
- ERD generation — Entity Relationship Diagrams rendered with Mermaid

**Data Viewer & Editor**
- Paginated table data viewer with deferred batch editing (Ctrl+S to commit staged changes)
- Inline add/delete rows; bulk multi-select operations; right-click context menus
- Data import from CSV and JSON with column mapping

**SQL Editor**
- Syntax highlighting (CodeMirror) with per-connection autocomplete (tables, columns, keywords)
- Multiple query tabs, query history, Explain/Analyze
- Streaming export of results to CSV, JSON, or SQL INSERT — written directly to disk, no renderer buffering
- Configurable row cap (default 10,000) with virtual scrolling on the result grid

**MongoDB**
- Database and collection browser
- Document viewer and editor (JSON tree view), insert/delete documents
- Aggregation pipeline runner

**Redis**
- Key browser with type-aware value viewer (string, list, set, hash, sorted set, stream)
- Set/edit/delete keys; built-in CLI passthrough

**App**
- Tabbed interface; dark/light theme with system preference detection
- Customizable keyboard shortcuts; saved query snippets
- Automatic update notifications with one-click install (via GitHub Releases)
- Cross-platform: Windows (`.exe` / `.msi`) and Linux (`.AppImage` / `.deb` / `.rpm`)

## Development

### Prerequisites

- Node.js 20+
- npm 10+

### Getting Started

```bash
# Clone the repository
git clone git@github.com:dsbroomeco/app-db-assistant.git
cd app-db-assistant

# Install dependencies
npm install

# Start the desktop app in development mode
npm run dev
```

### Project Structure

```
app-db-assistant/
├── build/                 # App icons (generated by scripts/generate-icons.js)
├── scripts/               # Build and utility scripts
├── docker/                # Dockerfiles for local CI pipeline testing
├── src/                   # Electron + React desktop app
│   ├── main/              # Electron main process
│   │   ├── main.ts        # App entry, window management, IPC handlers
│   │   └── preload.ts     # Context bridge for renderer
│   ├── renderer/          # React UI (Vite)
│   │   ├── components/    # React components (Sidebar, TabBar, TreeView, TableData, TableStructure, QueryEditor, MongoViews, RedisViews)
│   │   ├── context/       # React context providers (Connection, Settings, Theme)
│   │   ├── hooks/         # Custom hooks (useTabs)
│   │   ├── utils/         # Utilities (query result export)
│   │   ├── styles/        # Global CSS
│   │   └── App.tsx        # Root app component
│   ├── shared/            # Shared types and utilities
│   │   ├── ipc.ts         # Typed IPC channel definitions (50 channels)
│   │   └── types/         # Shared type definitions
│   └── db/                # Database connection drivers
│       ├── drivers/       # PostgreSQL, MySQL, SQLite, MSSQL, MongoDB, Redis drivers
│       ├── connection-manager.ts  # Connection lifecycle management
│       ├── credential-store.ts    # Encrypted credential storage
│       ├── ssh-tunnel.ts          # SSH tunnel management
│       ├── data-import.ts         # CSV/JSON import engine
│       ├── schema-diff.ts         # Schema comparison engine
│       ├── saved-queries.ts       # Saved query persistence
│       └── sanitize.ts            # Error message sanitization
├── tests/                 # E2E tests
├── AGENTS.md              # Agent coding instructions
├── ROADMAP.md             # Development roadmap
└── ARCHITECTURE.md        # Technical architecture
```

### Scripts

| Command              | Description                         |
| -------------------- | ----------------------------------- |
| `npm run dev`        | Start desktop app (dev mode)        |
| `npm run build`      | Build main process + renderer       |
| `npm run build:win`  | Build + package for Windows         |
| `npm run build:linux`| Build + package for Linux           |
| `npm run pack`       | Build + pack without installer (for local testing) |
| `npm test`           | Run unit tests                      |
| `npm run test:watch` | Run unit tests in watch mode        |
| `npm run test:e2e`   | Run end-to-end tests                |
| `npm run lint`       | Lint all source files               |
| `npm run lint:fix`   | Lint and auto-fix                   |
| `npm run typecheck`  | TypeScript type checking only       |
| `npm run release`    | Bump version, update changelog, create git tag |
| `npm run release:beta` | Create a beta pre-release          |
| `npm run release:dry` | Preview release without making changes |
| `docker compose run --rm build-linux` | Full Linux pipeline in Docker |
| `docker compose run --rm build-win` | Windows cross-compile in Docker |
